# Курсач: Приложение для построения триангуляции

## Описание
Приложение на Java с красивым интерфейсом для построения триангуляции заданной области (жадный алгоритм). Поддерживает ввод данных с клавиатуры и из файла, вывод результатов на экран и в файл.

## Функционал
- Ввод данных: с клавиатуры и из файла
- Вывод данных: на экран и в файл
- Визуализация триангуляции
- Жадный алгоритм триангуляции

## Используемые технологии
- Java
- JavaFX (UI)

## Запуск
1. Открыть проект в IDE (например, VS Code или IntelliJ IDEA)
2. Собрать проект
3. Запустить Main класс

## Структура проекта
- src/ — исходный код
- resources/ — ресурсы (иконки, стили)
